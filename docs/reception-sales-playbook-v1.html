NEUROVA İstek Kütüphanesi • Resepsiyon Satış Playbook v1.0

1) Satış Akışı

Amacımız: İhtiyaç → Öneri → Kapanış (rezervasyon)

  1. 60 saniyede 4 soru: hedef, kişi, süre, tarih/saat.
  2. Motor öneri üretir; 2 seçenek (ideal + alternatif).
  3. Kapanış: “Size şu saati ayırıyorum... Onaylıyor muyuz?”

“Düşünelim” cevabı → mini hold + belirli takip zamanı.

2) Satış Script’i (TR)

  • “Hoş geldiniz... 3 kısa soru sorayım.”
  • İhtiyaç: “Rahatlama mı toparlanma mı?”, “60 mı 90 mı?”, “Koku hassasiyetiniz var mı?”
  • Öneri: “En uygun [Paket A], Alternatif [Paket B].”
  • Kapanış: “Bugün ... veya ... uygunum. Hangisini ayıralım?”
  • Up-sell: “Hot Stone / Aroma / Head add-on ile etkiyi yükseltebiliriz.”

3) “Ürün + Paket” Kuralı

  1. Temel: tek uygulama (60 dk).
  2. Değer: kombin (Hamam+Masaj / Recovery set).
  3. Premium: Signature / Couple / Prestige + upgrade.

Kararsız misafirlerde değer paketi ile kapanış daha kolay.

4) Takip Sistemi

  • Etiketler: Lead kaynağı, niyet, durum, tek not (neden almadı?).
  • Hold sonrası 2 saat içinde kısa mesaj.
  • Aynı gün akşam “son hatırlatma”.
  • Ertesi gün “alternatif saat” öner.

5) KPI’lar

  1. Günlük lead sayısı
  2. Rezervasyona dönüşüm %
  3. Ortalama sepet (tek/kombin/premium)
  4. Upgrade oranı
  5. No-show / iptal oranı

6) İstek Kütüphanesi Kullanımı

  • Misafire 2 görsel + 2 seçenek göster (fazla yok).
  • WhatsApp’a “paket adı + 2 cümle + görsel” şablonu.
  • Dil otomatik seç (TR/EN filtrenize göre).
Operational model: One script, one flow, one tracking system → clear visibility for Reception sales.
