Resepsiyon Quick Use / Reception Quick Use

Standart Akış

  1. Rol seç: Reception
  2. Dil seç: misafire göre TR / EN
  3. Kategori seç: Hamam / Masaj / Kids & Family / Face / Recovery
  4. Kartı aç → metin aynen kopyala (değiştirme)
  5. Çıktıyı kontrol et → gerekirse 1 kez tekrar üret

Standard Flow (EN)

  1. Select role: Reception
  2. Select language: guest’s choice TR / EN
  3. Pick category: Hammam / Massage / Kids & Family / Face / Recovery
  4. Open the card → copy the text as-is (do not edit)
  5. Check the output → regenerate once if needed

8 Altın Kural

  • Therapist prompt’u kullanma
  • TR/EN karıştırma
  • “Daha seksi / daha iddialı” isteklerinde sınırı zorlama
  • Kids içeriklerinde SafeNote şart
  • Tam yüz / yakın plan istemiyoruz
  • Aynı prompt’u spam yapma (2–3 deneme)
  • Boş ekran → Ctrl+Shift+R
  • Emin değilsen → yöneticiye sor

8 Golden Rules (EN)

  • Do NOT use Therapist prompts
  • Keep TR and EN separate
  • Hold the line on “more sexy / more daring” requests
  • Kids content requires SafeNote
  • No full-face/close-up imagery
  • Don’t spam the same prompt (2–3 attempts max)
  • Blank screen → Ctrl+Shift+R
  • When unsure → ask your manager

TR Quick Replies

  • “Rahatlama mı, toparlanma mı, detoks mu?”
  • “Hamam + masaj kombin mi yoksa tek uygulama mı?”
  • “Lavanta / narenciye / kokusuz tercihiniz var mı?”
  • “Hangi saat uygundur? (sabah / öğlen / akşam)”

EN Quick Replies

  • “Deep relax, recovery, or detox?”
  • “Hammam + Massage combo or a single treatment?”
  • “Fragrance sensitivity? (lavender / citrus / fragrance-free)”
  • “What time suits you best? (morning / afternoon / evening)”
